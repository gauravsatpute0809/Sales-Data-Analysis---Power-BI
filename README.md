# 📊 Power BI Sales Data Analysis Project

## 🧾 Overview

This project is a Sales Data Analysis Dashboard created in Power BI using an Excel dataset. The goal of this project was to convert raw sales data into useful business insights through interactive reports and visualizations.

The dashboard helps understand product performance, sales growth, profit trends, customer orders, discounts, and city-wise sales performance.

---

## 🛠️ What I Worked On

During this project, I covered the complete Power BI workflow:

* 📥 Imported and cleaned raw data in Power BI
* 🔍 Performed data profiling and transformations using Power Query
* 🔗 Built relationships between tables using data modeling concepts
* 🧩 Applied Primary Key, Foreign Key, Cardinality, and Star Schema
* 📐 Created calculated measures using DAX
* 📊 Designed an interactive dashboard with filters and slicers

---

## 📌 Dashboard Requirements Covered

* 🥇 Top 5 and Bottom 5 Products by Sales, Profit, and Quantity Sold
* 📈 Sales trends over time (Daily, Monthly, Quarterly, Yearly)
* 🔄 Sales vs Profit relationship analysis
* ⚖️ Comparison between two selected periods
* 💸 Average discount by discount category
* 🧾 Total number of orders
* 📋 Order-level report with filters
* 🌍 Sales by different cities
